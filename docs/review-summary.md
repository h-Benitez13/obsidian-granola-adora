# Review Summary

The review summary is a lightweight digest for:

- pending recommendation queue items
- incident records available in the vault

It is designed for review workflows, not deep analysis.

## Current sections

- Overview counts
- Pending Recommendations
- Incidents

## Empty-state behavior

If no recommendations or incidents are present, the summary still renders with
explicit empty-state bullets so automation and reviewers can distinguish “no
data” from “failed to render.”
